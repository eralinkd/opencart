# OpenCart - Админ панель

Проект очищен от каталога (фронтенда), оставлена только админ-панель и ключевые элементы.

## Структура проекта

```
opencart/
├── admin/              # Админ-панель (полностью сохранена)
│   ├── controller/     # Контроллеры
│   ├── model/          # Модели
│   ├── view/           # Шаблоны и ресурсы
│   ├── language/       # Языковые файлы
│   ├── config.php      # Конфигурация админки
│   └── index.php       # Точка входа в админку
├── system/             # Системные файлы (нужны для работы)
├── image/              # Изображения (используется админкой)
├── extension/          # Расширения
└── error.html          # Страница ошибок
```

## Ключевые элементы авторизации

### Контроллеры (`admin/controller/common/`)
- **login.php** - Вход в систему
- **logout.php** - Выход из системы
- **forgotten.php** - Восстановление пароля
- **authorize.php** - Двухфакторная авторизация

### Шаблоны (`admin/view/template/common/`)
- **login.twig** - Форма входа
- **forgotten.twig** - Форма восстановления пароля
- **forgotten_reset.twig** - Сброс пароля
- **authorize.брос двухфакторной авторизации
- **authorize_unlock.twig** - Рtwig** - Форма двухфакторной авторизации
- **authorize_reset.twig** - Сазблокировка аккаунта

### Основные компоненты админки
- **header.php** / **header.twig** - Шапка админки
- **footer.php** / **footer.twig** - Подвал админки
- **dashboard.php** / **dashboard.twig** - Главная панель
- **column_left.php** / **column_left.twig** - Левое меню

## Доступ к админке

URL: `http://localhost:8080/opencart/admin/`

## Удалено

- ❌ `catalog/` - Папка каталога (фронтенд магазина)
- ❌ `index.php` - Точка входа каталога
- ❌ `config.php` - Конфигурация каталога
- ❌ `cron.php` - Cron задачи каталога

## Что осталось

- ✅ `admin/` - Полная админ-панель
- ✅ `system/` - Системные файлы (обязательны)
- ✅ `image/` - Изображения
- ✅ `extension/` - Расширения
- ✅ Ключевые элементы авторизации

## Использование

Все функции админки работают как обычно. Можно использовать элементы авторизации как примеры для изучения:
- Система логина/логаута
- Восстановление пароля
- Двухфакторная авторизация
- Работа с сессиями и токенами
